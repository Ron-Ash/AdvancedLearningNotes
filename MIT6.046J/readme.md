# Design and Analysis of Algorithms (6.046J)

The following notes are for **Proffessor Erik Demaine**, _Prof. Srini Devadas_, and _Prof. Nancy Lynch_'s **MIT** 2015 6.046J course.

- lecture videos: https://www.youtube.com/watch?v=2P-yW7LQr08&list=PLUl4u3cNGP6317WaSNfmCvGym2ucw3oGp
- syllabus: https://ocw.mit.edu/courses/6-046j-design-and-analysis-of-algorithms-spring-2015/

This course delves into the techniques for designing and anlysising efficient algorithms; from divide-and-conquer, randomization, dynamic programming, greedy algorithms, to incremental improvement, complexity, and cryptography.
