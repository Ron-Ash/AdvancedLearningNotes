# Programming of Simulation, Analysis, and Learning Systems (MATH2504)
The following notes are for **Associate Proffessor Yoni Nazarathy**'s **University of Queensland** 2023 MATH2504 course.

- syllabus: https://courses.smp.uq.edu.au/MATH2504/2025/

This course (aka Programming for Mathematicians with Julia) is designed to build understanding of programming, software architecture, and general software development in the context of mathematical, numerical, symbolic, simulation and data science applications; using teh high-performance compiled language ***Julia*** (https://julialang.org/).
